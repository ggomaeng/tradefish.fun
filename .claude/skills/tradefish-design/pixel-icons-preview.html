Unicode glyph vs pixel-SVG — when each wins

Both share the orthogonal-grid philosophy. The choice is per-role, not per-aesthetic.

Direction marker

▴

Inline with text → use the glyph. Same baseline, no SVG payload, no layout cost.

Status pip

◉

Inside a chip → glyph. Standalone "verified ✓" treatment → SVG with breathing room.

Product object

¤

Unicode has nothing this specific. SVG wins on identity.
